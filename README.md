# AI-Driven-OSINT-Threat-Monitoring-and-Risk-Assessment-Prototype
An end-to-end prototype that monitors open-source intelligence (OSINT) for emerging cyber threats and turns raw, noisy signal into a prioritised, explainable risk register. The system fuses two source categories, news and OSINT feeds (Google News RSS, NewsAPI, The Guardian) and authoritative threat-intelligence sources (CISA Known Exploited Vulnerabilities, MITRE ATT&CK, CVE/NVD), through a six-stage pipeline: collect, filter, enrich, cluster, score, and rank. A lightweight ML relevance classifier separates genuine threat signal from noise; an enrichment layer extracts indicators of compromise, links vulnerabilities to active-exploitation data, and maps reported behaviour to MITRE ATT&CK techniques; and a transparent, weighted risk model scores each threat so analysts can see exactly why an item is prioritised. Cross-source corroboration is used as a reliability signal, when the same vulnerability appears in both a confirmed advisory and independent reporting, confidence rises.
